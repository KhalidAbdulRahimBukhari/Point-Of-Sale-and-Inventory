# POS & Inventory Management System

## Live Demo
👉 https://point-of-sale-and-inventory.vercel.app/

## Description
A full-stack Point of Sale (POS) and Inventory Management System built for a single-store retail environment.  
The system handles sales transactions, inventory tracking, invoice generation, and business analytics through a modern web interface.

### Key Features
- POS system with barcode support  
- Percentage-based discounts  
- Invoice generation and history  
- Inventory and stock management  
- Returns and refunds handling  
- Dashboard with charts and tables  
- JWT authentication with Admin/Cashier roles  

### Tech Stack
- ASP.NET Web API  
- SQL Server  
- React + Vite + TypeScript
